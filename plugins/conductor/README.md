# conductor

Context-Driven Development plugin that transforms Claude Code into a project management tool with structured workflow: Context → Spec & Plan → Implement

Converted from Claude plugin source to Copilot plugin layout.

## Commands

| Name | Description |
|---|---|
| `implement` | /conductor:implement prompt |
| `manage` | /conductor:manage prompt |
| `new-track` | /conductor:new-track prompt |
| `revert` | /conductor:revert prompt |
| `setup` | /conductor:setup prompt |
| `status` | /conductor:status prompt |
